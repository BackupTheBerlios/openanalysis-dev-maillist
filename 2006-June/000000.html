Andy,

Congratulations!!  Your OpenAnalysis changes passed regression  
testing in UseOA-Open64.

I just ran the UseOA-Open64 regression tests on rev 108, rev 111, and  
rev 115 of OpenAnalysis.  Each of them involved significant changes.   
Here is an outline of the process:

	In OpenAnalysis
		% svn update -r 108
		% make -f Makefile.quick configure
		% make -f Makefile.quick install

	In UseOA-Open64/
		% make clean
		% make
		% source scripts/regressAll.script


Not all of the updates required a configure of OpenAnalysis.  The  
make clean is because the makefile dependencies in UseOA-Open64 (and  
UseOA-ROSE) are not that robust and I wanted to be sure all changes  
to header files are compiled in.

We don't have solid regression tests up and running yet for UseOA- 
ROSE, but we are very close right Andy? =)
