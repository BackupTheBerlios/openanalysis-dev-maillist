Hi All,

OpenAnalysis and UseOA-ROSE have a new build system.  The major
difference is that environmental variables for OA projects are now
prefixed with the projectname.  For example: CXXFLAGS is now
OA_CXXFLAGS for OpenAnalayis and USEOA_ROSE_CXXFLAGS for UseOA-Rose.
Also, the makefile will output a build.log file describing the state
of these variables when run.

I've made these changes since having unique names for each project
made writing a series of scripts I did for benchmarks easier.  I think
organizing it this way will prove useful in the long run.

I've provided a script (available for download at the WIKI) for
configuring all of these environmental variables.  I suggest you
download it, configure it, and source it in your .bashrc or .cshrc.
See the page on the WIKI about it at:
http://www-unix.mcs.anl.gov/OpenAnalysisWiki/moin.cgi/DeveloperScript

If you're using Okapi see the notes on the WIKI at:
http://www-unix.mcs.anl.gov/OpenAnalysisWiki/moin.cgi/NewBuildNotes

Thanks,
-Andy
