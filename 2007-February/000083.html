Installing OpenAnalysis and UseOA-ROSE

Look at the INSTALL files for OpenAnalysis and UseOA-ROSE.

Here are the environment variable definitions for building UseOA-ROSE  
with ROSE-0.8.9a on the CSU machines.

setenv PATH /usr/local/gcc-4.0.3/bin

setenv ROSEDIR /s/bach/e/proj/oadev/ROSE/rose-0.8.9a/armstrong-gcc403

setenv OADIR /wherever/you/put/it

ROSE-0.8.10a is installed using g++ 4.1.1, but UseOA-ROSE needs to be  
modified to work with it.
