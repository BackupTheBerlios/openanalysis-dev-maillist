I recently had Luis put all of the Open64IRInterface stuff into a  
standalone package that builds into a library.  I just made UseOA- 
Open64 so that it uses the standalone Whirl2OA package.  I am still  
working on cleaning up the build process for UseOA-Open64, but  
everything works with regressAll right now.  The key thing to  
remember is that to make the most recent version of UseOA-Open64  
compile, you need to get Whirl2OA.

in UseOA-Open64/
	% svn co svn+ssh://svn.berlios.de/svnroot/repos/openanalysis/ 
Whirl2OA/trunk Whirl2OA

If you have any problems let me know.  Please do not mess with the  
Makefiles because I am still working on them.
