Hi all,

Python on Okapi needs a minimum of GCC-3.3.*

Since we are using GCC-3.2 on Okapi with UseOA-Open64, you
cannot run scripts/genScripts.py in a terminal whose
environment has already been set up for compiling UseOA-Open64
on Okapi.

However, I was successful in running python from a fresh
terminal window on Okapi (whose environment had not been set
for UseOA-Open64) whose default GCC is GCC-3.4

I tried running python in a new tcsh shell in the same terminal
window whose environment had previously been set for
UseOA-Open64, but it must inherit the environment, as it gave
the same original GCC-3.3 needed error message.

-- Barbara
