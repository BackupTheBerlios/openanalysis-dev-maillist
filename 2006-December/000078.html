Hello all,

I am writing GettingStarted documentation, and the build process  
seems to be too complex and intertwined among all of the projects.  I  
can see having an all inclusive script for developers, but users  
should be able to just set some environment variables, configure, and  
make.  Therefore, changed the Makefiles in UseOA-ROSE and  
OpenAnalysis so that they look for the typical names (eg. CXXFLAGS),  
if the more specific name is not defined (eg. OA_CXXFLAGS).  I also  
updated the README and INSTALL files for both projects.  UseOA-Open64  
is in progress.  I am having problems with my Open64 installation.
