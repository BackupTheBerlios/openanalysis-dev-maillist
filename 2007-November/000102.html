Hi all,

I've just committed to the OpenAnalysis trunk (several times),
specifically revisions 533 through 539. The changes made are described
below:

Bug Fixes
-------------
- ManagerFIAlias has a MRE visitor class VisibleBaseVisitor. I've
modified it so that whenever an AdressOf node is visited the

visitor will return that the base is not visible. This is important
since an expression like (&A[32])[2] does not have a

visible base.
- During a backward dataflow analysis the atDGraphNode function should
do a meet of all in information from successors.

Changes so that I can do ExprTree ordering
---------------------------------------------
- Added operator< to the ConstValBasicInterface
- Added OpBasicInterface
- Added the NewExprTree class (In the NewExprTree/ dir)

Changes for DFAGen
------------------
- Added IR Interfaces to DFAGen'd ReachingDefs and Liveness analyses
- Added DFAGen/Liveness and DFAGen/ReachingDefs/ directories
- Added a succinct flag (defaulted to off) to the dumpLocSets (in
Location.hpp|cpp) that dumps the sets in a cleaner way.
- Added DFAGenDFSet class

- Modified Makefiles to compile added classes.

Thanks,
-Andy Stone
