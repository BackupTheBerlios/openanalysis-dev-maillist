Hi All,

I've made some updates to UseOA-ROSE that  add some nifty new debugging options.

 - Added support for a USEOA_DEBUG environmental variable (for more info
      look at the new debug.h file).
    - added the --skipAnalysis and --exitWithTop command line options.
 If       --skipAnalysis is set then when running when
--oa-FIAliasAliasMap the       analysis will be skipped (the other
analyses should be configured to take into account this flag as well).
 --exitWithTop will cause OATest to execute the unix 'top' command
immediately before termination.

Thanks,
