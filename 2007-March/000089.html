Hi,
 
If you see a version mismatch error after executing automake, just check that 
automake and aclocal are at the same version(1.9.6).

If not, you need to regenerate the script.
Executing autoconf does not solve the version mismatch error.
You need to type in the following commands to fix it:
>aclocal
>automake
